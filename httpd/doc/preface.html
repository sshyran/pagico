System requirements

Abyss Web Server is a multiplatform software designed for Microsoft Windows 95, 98, Millennium Edition (ME), NT 3.51 (and higher), 2000, XP (all editions), 2003, Vista (all editions), Mac OS X 10.2 (and higher), Linux 2.0 (and higher), and FreeBSD 6.0 (and higher) systems.

This guide documents the Linux version. Abyss Web Server for Linux minimum requirements are:

  • An Intel 80486 processor at 33 MHz.

  • 4 Mb of free RAM.

  • A modern web browser with HTML 4.0 and CSS 1.0 support.

  • TCP/IP protocol stack installed.

  • GLIBC 2.1 or higher.

Although Abyss Web Server can run on a standalone computer, it is recommended to have a Modem or a network card and to be connected to a LAN (Local Area Network) or to the Internet.
